# 동물농장: 혁명의 서곡 — 게임 기획안

## 개요

조지 오웰의 **동물농장(Animal Farm)** 을 기반으로 한 탑다운 로그라이크 생존 슈터. **궁수의 전설** 스타일의 직관적 조작과 중독성 있는 루프를 채용하고, 동물농장 특유의 풍자와 서사를 게임 메커니즘에 녹여낸다.

- **장르**: 탑다운 아레나 로그라이크 슈터
- **플랫폼**: 모바일 웹 (Next.js + Phaser 3)
- **세션 길이**: 10~12분 / 1판
- **타겟**: 15~35세, 궁수의 전설 플레이어, 인디/스토리 게임 팬

> **상세 설계 문서**: [CHARACTERS.md](./CHARACTERS.md) (캐릭터·스프라이트), [SATIRE.md](./SATIRE.md) (풍자·이스터에그)

## 디자인 철학 — 3박자

```
[외형] 스타듀밸리풍 아기자기한 픽셀 동물들
   ↓ 대비
[내용] 동물농장의 배신·폭력·권력 비판
   ↓ 양념
[병맛] AI 버블, 자본주의, 현대 정치 풍자 + 이스터에그
   ↓ 결과
[기억] "귀여운데 왠지 찝찝한 게임"
```

**시그니처 무드**: `귀여움 ⊕ 잔혹함 ⊕ 풍자`. 모든 시스템 메시지를 병맛 위트로 교체한다.
- Critical → **"너 정말 핵심을 찔렀어."**
- Level Up → **"계급 상승! (진짜였냐?)"**
- Game Over → **"혁명은 실패했다. 다시 시도하시겠습니까?"**

## 핵심 중독성 설계

궁수의 전설이 전 세계 1억 다운로드를 기록한 핵심 공식: **"하기 쉬운데, 끝내기 어렵다"**. 여기에 **"한 판 더"**를 만드는 5가지 축.

| 축 | 원리 | 동물농장 적용 |
|---|---|---|
| **변수성** | 매 판 다른 경험 | 무작위 업그레이드 3지선다, 적 구성 무작위 |
| **영구 성장** | 죽어도 강해진다 | 탤런트 트리, 캐릭터 해금, 장비 업그레이드 |
| **근접 성공** | "아깝다!" 유도 | 보스 HP 10% 남기고 사망 → 광고 부활 or 즉시 재도전 |
| **수집 욕구** | 다 모으고 싶다 | 캐릭터 4종, 유물 20종, 진엔딩 3종 |
| **숏 세션** | 10분이면 한 판 | 웨이브 10~20개, 점심시간/출퇴근길에 최적 |

## 세계관 & 스토리 구조

### 제1장: 혁명 — Manor Farm 탈출
_"모든 동물은 평등하다"_

존스 농부의 학대를 견디다 못해 동물들이 반란을 일으킨다. 플레이어는 동물 혁명군의 선봉으로 농장에서 탈출해 자유의 땅을 향해 나아간다.

- **적**: 농장 인부, 사냥개, 감시자
- **보스**: 농부 존스 (산탄총 + 사냥개 소환)
- **웨이브**: 10

### 제2장: 건설 — 풍차의 꿈
_"두 발로 걷는 것은 나쁘다"_

동물 농장이 세워지고, 스노볼이 설계한 풍차 건설이 시작된다. 그러나 이웃 농장 프레드릭이 군대를 이끌고 침공해 온다.

- **적**: 인간 침입자, 방화범, 배신자 양
- **보스**: 프레드릭 (폭탄 투척 + 화염 공격)
- **웨이브**: 15

### 제3장: 배신 — 나폴레옹의 독재 ⭐ 핵심 스토리
_"모든 동물은 평등하다. 그러나 어떤 동물은 더 평등하다"_

나폴레옹이 권력을 장악하고 비밀경찰을 동원해 반대파를 숙청한다. 플레이어는 나폴레옹의 독재에 맞서 저항군의 마지막 희망이 된다.

- **적**: 비밀경찰견, 선전원 돼지, 세뇌된 양떼
- **보스**: 나폴레옹 (스킬 카피 + 독재 오라)
- **웨이브**: 20

### 제4장: 해방 — 진실의 서곡 🔒 숨겨진 챕터
_"동물들이 창문 너머를 바라보았다"_

조작된 7계명을 파괴하고 진정한 평등을 되찾기 위한 마지막 전투.

- **해금 조건**: 3장 A랭크 이상 클리어
- **적**: 모든 보스의 망령, 강화된 비밀경찰, 인간화된 돼지들
- **보스**: 인간-돼지 연합 (나폴레옹 + 필킹턴 동시 등장)
- **웨이브**: 25

## 캐릭터 5+1인

각 캐릭터는 고유한 **픽셀아트 스프라이트**, 능력치, 패시브, 액티브 스킬 4개를 보유한다. 스프라이트 상세 명세는 [CHARACTERS.md](./CHARACTERS.md) 참고.

### 스노볼 (돼지) — 전략가/지휘관
- **해금**: 기본 | **스프라이트**: 32×32, 흰돼지 + 빨간밴다나
- **능력치**: 공속 ↑, 이속 ↓
- **패시브 "7계명"**: 7번째 공격마다 2배 데미지
- **액티브 스킬**:
  - Q: **혁명의 돌격** — 전방으로 돌진하며 경로상 적에게 피해
  - W: **풍차 설계도** — 5초간 공격력 30% 증가하는 버프 존 생성
  - E: **연설의 힘** — 주변 적 3초간 이동/공격속도 50% 감소
  - R: **전술 지휘** — 8초간 공격력 50% 증가

### 복서 (말) — 탱커/돌격
- **해금**: 기본 | **스프라이트**: 32×32, 갈색말 + 붕대 감은 다리
- **능력치**: HP ↑↑, 공격력 ↑, 이속 ↓↓
- **패시브 "더 열심히"**: HP 50% 이하 시 공격력 50% 증가
- **액티브 스킬**:
  - Q: **돌진** — 전방으로 천천히 이동하며 지속 피해
  - W: **인내** — 3초간 받는 피해 50% 감소
  - E: **말발굽 강타** — 근접 범위에 큰 피해
  - R: **"더 열심히 일하겠습니다"** — 10초간 공격력 100% 증가, 이후 반동 페널티

### 벤자민 (당나귀) — 회피/카운터
- **해금**: 1장 클리어 | **스프라이트**: 28×32, 회색 당나귀 + 허리에 책
- **능력치**: 이속 ↑, HP ↓
- **패시브 "회의주의"**: 회피 성공 시 2초간 무적
- **액티브 스킬**:
  - Q: **회피 기동** — 짧은 대시, 잔상이 적 어그로 분산
  - W: **통찰** — 적 체력바 표시 + 치명타율 증가
  - E: **냉소** — 주변 적 이속/공격력 감소
  - R: **마지막 통찰** — 시간 느려짐 + 공속 폭발적 증가

### 나폴레옹 (돼지) — 독재자 (플레이어블)
- **해금**: 3장 클리어 | **스프라이트**: 32×32, 검은돼지 + 중절모 + 담배
- **능력치**: 마나↑, HP 중간
- **패시브 "권력의 맛"**: 적 처치 시 공격력 중첩 증가
- **액티브 스킬**:
  - Q: **비밀경찰 소환** — 충성스러운 개 2마리 소환
  - W: **선전** — 적 세뇌하여 아군으로 전환
  - E: **배후 찌르기** — 적 뒤로 순간이동 후 강력한 일격
  - R: **7계명 선포** — 화면 전체 피해 + 처치 시 폭발 연쇄

### 몰리 (암말) — 도망자/회복
- **해금**: 2장 클리어 | **스프라이트**: 28×32, 백마 + 분홍리본 + 설탕덩어리
- **능력치**: 이속 ↑↑, HP ↓↓
- **패시브 "리본"**: 적 처치 시 HP 5% 회복
- **액티브 스킬**:
  - Q: **도망쳐!** — 빠른 속도로 먼 거리 대시
  - W: **달콤한 유혹** — 적 유인하는 디코이 설치
  - E: **설탕 덩어리** — HP 30% 즉시 회복
  - R: **인간과의 거래** — 8초간 이속 60% 증가 + 회피 50% 증가

### 🐑 양 (Sheeple) — 숨겨진 캐릭터
- **해금**: 적 양 1000마리 처치
- **능력치**: 이속↓, HP 중간, **스턴 특화**
- **패시브 "양떼 효과"**: 주변 양 개체당 공속 10% 증가
- **액티브 스킬**:
  - Q: **"네 다리는 좋다"** — 주변 적 2초 스턴 (자신도 1초 스턴)
  - W: **집단 행동** — 화면 내 모든 양 떼가 플레이어에게 집결 + 돌진
  - E: **메에에~** — 4초간 적 이속 60% 감소 + 혼란
  - R: **대이동** — 맵 전체 양떼 소환, 8초간 적과 충돌 시 피해

## 전투 시스템

### 기본 조작
| 입력 | 액션 |
|---|---|
| 한 손가락 드래그 | 캐릭터 이동 |
| 정지 | 가장 가까운 적 자동 조준 + 자동 발사 |
| 더블 탭 / Space | 대시 (쿨다운 3초) |
| 하단 버튼 Q/W/E/R | 액티브 스킬 발동 |
| 좌측 상단 버튼 | 일시정지 |

### 웨이브 구조
```
웨이브 1~N-1: 일반 적 웨이브 (난이도 점진적 상승)
  ├─ 모든 적 처치 시 클리어
  ├─ 3지선다 업그레이드 선택 (무작위 3개 중 1개)
  ├─ HP 20% 회복
  └─ 5웨이브마다 정비 상점 (ideaFragments 소모)

마지막 웨이브: 보스 전투
  ├─ 보스 전용 HP 바 표시 (화면 상단 중앙)
  ├─ 보스 패턴: 일반 공격 → 특수 패턴 → 광폭화 (HP 30% 이하)
  └─ 클리어 시 대량 보상 + 챕터 클리어
```

### 3지선다 업그레이드 (핵심 중독 루프)

웨이브 클리어마다 무작위 3개 중 1개를 선택. 선택한 업그레이드는 해당 판에만 적용되며, 중첩 선택 시 효과가 강화된다.

| 등급 | 확률 | 예시 |
|---|---|---|
| 일반 (흰색) | 45% | 공격력 +10%, 이속 +15%, HP +30 |
| 희귀 (파란색) | 30% | 발사체 +1, 투사체 속도 +30%, 관통 +1 |
| 영웅 (보라색) | 18% | 전방 부채꼴 발사, 후방 공격, 치명타 확률 30% |
| 전설 (황금색) | 7% | 대각선 발사, 발사체 2배, 매 10초 무적 2초 |

#### 업그레이드 시너지 (특정 조합 달성 시 추가 효과)
- **7계명의 진실**: 전방 부채꼴 + 대각선 공격 + 후방 공격 → 전방향 발사
- **복서의 분노**: 공격력 +30% + 공속 +30% + 치명타 → 3초간 공격력 3배
- **벤자민의 통찰**: 이속 20% + 대시 쿨 감소 → 대시 잔상이 데미지 판정
- **몰리의 배신**: 회복량 +20% + 디코이 지속시간 → 처치 시 2초 무적

## 메타 프로그레션 (영구 성장)

### 탤런트 트리
캐릭터 공통 영구 업그레이드. `ideaFragments`를 소모해 레벨업한다.

#### 기초 탤런트 (각 6레벨)
| 탤런트 | 효과 (레벨당) | 최대 |
|---|---|---|
| 공격력 | +5% | +30% |
| 최대 HP | +10% | +60% |
| 공격 속도 | +5% | +30% |
| 이동 속도 | +3% | +20% |
| 대시 쿨타임 | -5% | -30% |

#### 고급 탤런트 (계정 레벨 20 해금, 각 3레벨)
| 탤런트 | 효과 (레벨당) | 최대 |
|---|---|---|
| 치명타 확률 | +3% | +10% |
| 치명타 배율 | +20% | +80% |
| HP 회복량 | +10% | +40% |
| 발사체 관통 | +1 | +3 |
| 보스 추가 피해 | +10% | +40% |

### 수집 요소
- **유물 20종**: 특정 조건 달성 시 해금 (예: "오웰의 타자기" = 모든 챕터 S랭크)
- **장비 4슬롯**: 무기 / 방어구 / 장신구 / 특수 (각 5티어, 아이디어 합성으로 획득)
- **진엔딩 3종**:
  1. **혁명의 완수** — 나폴레옹 타도 (4장 클리어)
  2. **타협의 길** — 나폴레옹과 협상 (3장에서 특정 조건)
  3. **침묵의 진실** — 모든 7계명 조각 수집

| 인간 침입자 | 2장 | 50 | 원거리 총격, 엄폐 |

| 배신자 양 | 2장 | 40 | 자폭 돌진 |
| 비밀경찰견 | 3장 | 80 | 3연속 돌진 + 출혈 디버프 |
| 선전원 돼지 | 3장 | 60 | 주변 적 버프 + 도주 |
| 세뇌된 양떼 | 3장 | 20 | 다수 동시 스폰, 물량 압박 |

### 보스

#### 농부 존스 (1장)
- **산탄총**: 전방 3방향 발사
- **사냥개 소환**: 2마리 소환
- **광폭화 (HP 30%)**: 연속 산탄총 + 이동속도 50% 증가

#### 프레드릭 (2장)
- **폭탄 투척**: 3초 후 폭발하는 지뢰 설치
- **화염병**: 바닥에 지속 피해 장판 생성
- **광폭화 (HP 30%)**: 화염 장판 2배 면적 + 폭탄 3개 동시 투척

#### 나폴레옹 (3장)
- **스킬 카피**: 플레이어의 마지막 사용 스킬을 복제해 사용
- **독재 오라**: 5초마다 주변 적 공격력 20% 증가 버프
- **비밀경찰 소환**: 비밀경찰견 3마리 소환
- **광폭화 (HP 30%)**: "모든 동물은 평등하다 — 그러나 어떤 동물은 더 평등하다" — 전체 화면 장판

#### 인간-돼지 연합 (4장)
- **나폴레옹 + 필킹턴 동시 등장**
- **나폴레옹**: 독재 오라 + 스킬 카피
- **필킹턴**: 원거리 저격 + 함정 설치
- **연계 패턴**: "거래" — 두 보스가 HP를 30%씩 공유 회복
- **광폭화 (합계 HP 30%)**: 동시 광폭화 + 맵 전체 장판

## AI 합성: 이데아의 조각

기존에 설계된 AI 합성 시스템을 게임 루프에 통합한다.

- **ideaFragments**: 적 처치 시 획득 (기본 1~3개, 보스 50개)
- **100개 소모**: AI 합성으로 무작위 장비/유물 획득
- **캐릭터 + 챕터 조합**: 특정 조건에서 전용 아이템 출현
- **예: 나폴레옹 캐릭 + 3장 = "독재자의 유언" (전용 유물)**

## 중독성 심리 분석 — 왜 100시간을 박는가

```
[입문기] 첫 판은 누구나 1장 절반까지 진행 가능
  → "나도 할 수 있네?" 자신감 부여
  → 즉각적인 성취감 (웨이브 5~6까지 무난하게 진행)

[몰입기] 3지선다 업그레이드 선택
  → "이번 판은 무슨 빌드가 나올까?" 기대감
  → 전략적 선택권 + 운 요소의 조화
  → "아까 그 조합 다시 해보고 싶다" → 재도전 유도

[근접 실패] 보스 HP 10~30%에서 사망
  → Frustration → "다음엔 깬다!" 결의
  → 1회 광고 부활 선택지로 이탈 방지
  → "이번 판 업그레이드 빌드가 좋았는데..." → 유산 심리

[영구 성장] 죽어도 ideaFragments는 쌓임
  → "5개만 더 모으면 탤런트 업그레이드!"
  → 절대 후퇴하지 않는 진보감
  → "다음 판은 더 강해져서 시작한다"

[시너지 발견] 업그레이드 조합의 즐거움
  → "전방위 발사 + 관통 + 바운스 = 화면이 정리된다!"
  → 유저가 직접 발견하는 기쁨
  → 커뮤니티 공략 공유로 바이럴 마케팅

[수집 충동] 13/20 유물, 1/3 엔딩
  → 도감의 빈 칸이 불편하다
  → "몇 개만 더 모으면 100%!"
  → 희귀 유물 획득 시 도파민 폭발

[숏 세션] 10분이면 한 판
  → "한 판만 더..." → 어느새 2시간
  → 대중교통, 점심시간, 침대 위 등 틈새 시간에 최적화
  → 앱 전환 없이 모바일 웹에서 즉시 실행
```

## 기술 아키텍처

### 기술 스택
| 계층 | 기술 |
|---|---|
| 게임 엔진 | Phaser 3 (Arcade Physics) |
| UI 프레임워크 | Next.js 15 (App Router, Static Export) |
| 랜더링 | React 19 + HTML/CSS 오버레이 |

| 저장소 | localStorage (영구 세이브) |

추가:
