# SATIRE.md — 병맛·풍자·이스터에그 기획

## 기조

> "대중은 개돼지다" — 이 냉소적 명제를 **귀여운 픽셀로 감싸서** 독이 든 사탕처럼 전달한다.

동물농장 원작의 풍자가 1945년 소련 비판이었다면, 이 게임은 **2026년 대한민국 + 글로벌 테크 자본주의**를 비튼다.

## 1. 커스텀 텍스트 — 시스템 메시지 전면 교체

모든 게임 내 시스템 메시지를 병맛 위트로 대체한다.

| 원문 | 대체 텍스트 |
|---|---|
| Critical Hit! | **"너 정말 핵심을 찔렀어."** |
| Level Up! | **"계급 상승! (진짜였냐?)"** |
| Game Over | **"혁명은 실패했다. 다시 시도하시겠습니까?"** |
| Wave Cleared! | **"이 파도는 넘었다. 하지만 진짜 파도는..."** |

| Boss Defeated! | **"독재자를 타도했다. 근데... 다음은 누구?"** |

## 2. 이스터에그 — 원작 명대사 발동 조건

### "두 다리는 나쁘고, 네 다리는 좋다"

| 조건 | 발동 |
|---|---|
| 양 캐릭터로 보스 처치 | 화면 전체를 울리는 메아리 텍스트 (3초) |
| 웨이브 4 클리어 (4 = 네 다리) | "네 다리가 좋은 이유: 웨이브 4 클리어." 부연 설명 |
| 적 양 10마리 연속 처치 | "양들이 당황하고 있다. 네 다리로 서 있을 수 없어서." |
| 두 다리로 걷는 인간형 보스 조우 | "경고: 두 다리 보행 개체 발견." (경고창 UI) |

### "모든 동물은 평등하다. 그러나 어떤 동물은 더 평등하다"

| 조건 | 발동 |
|---|---|
| 나폴레옹으로 다른 보스 처치 | "보다시피, 나는 '더' 평등한 동물이었다." |
| 모든 캐릭터 해금 | "이제 모두 평등해. 진짜로." (아이러니 업적) |
| 치트키 입력 시 | "어... 어떤 동물은 코드로 더 평등하군." |
| 같은 적 100회 처치 | "통계적으로 평등하지 않은 살육 분포입니다." |

### "더 열심히 일하겠습니다" (복서 시그니처)

| 조건 | 발동 |
|---|---|
| 복서 HP 30% 이하 | 자동 대사 + 공격력 50% 버프 |
| 연속 3판 플레이 (쉬지 않음) | 로비에서 "쉴 틈이 어디 있나요. 더 열심히 일하겠습니다." |
| 하루 최대 접속 달성 | "오늘도 수고했다, 복서. (하지만 내일도 일해야 한다)" |

## 5. 자본주의 vs 공산주의 — 게임 메커니즘으로 비틀기

### 분배 시스템 (자본주의 모드 vs 공산주의 모드)

3장 이후 선택 가능한 난이도 변경:

| 모드 | 룰 | 병맛 설명 |
|---|---|---|
| **자본주의 모드** | ideaFragments 2배 획득, but 자신 HP 70% 감소 | "시장은 자유롭다. 죽음도 자유다." |
| **공산주의 모드** | 모든 적 HP 50% 감소, but ideaFragments 70% 감소 | "평등하게 나누면 다 같이 굶는다." |
| **혼합경제** (기본) | 기본 그대로 | "결국 이게 답이더라." |

### 가차 시스템 풍자

```
[상점 UI]
"확률형 아이템: 0.3% 확률로 전설 무기"
※ 실제 확률: 0.299999...% (반올림했습니다)
※ 확률 조작 의혹은 사실이 아닙니다. (총괄 디렉터: 나폴레옹)
```

## 6. Sheeple (양떼 민중) 시스템

### 양 적 AI의 특수 행동

```typescript
// Enemy.ts 확장 — Sheeple 타입
{
  type: "sheeple",
  behavior: "follow_leader", // 주변 적 리더를 따라 움직임
  special: "four_legs_chant", // 5초마다 "네 다리는 좋다!" 주변 적 이속 버프
  flee_when_leader_dies: true, // 리더 사망 시 공황상태로 이탈
  convert_on_napoleon: true, // 나폴레옹 플레이 시 10% 확률로 아군 전환
}
```

### 숨겨진 히든 보스: "빅 브라더 양"

> 4장에서 양을 단 한 마리도 죽이지 않으면 등장하는 평화주의 루트 보스.
> 거대한 양의 형상에 눈이 수백 개. 대사: "나는 너를 지켜보고 있었다. 네 다리는... 좋았다."

### 양떼 관련 병맛 시스템 메시지

```
"또 양이군. 근데 얘네 왜 이렇게 많은 거지?"
"양 1마리 추가... 1마리 추가... 1마리 추가..."
"양이 너무 많다. 이게 민주주의인가 독재인가."
"양떼를 이끄는 건 늑대가 아니라 돼지였다 — 아이러니."
```

## 8. 스토리 분기 — 병맛 엔딩

### 메인 엔딩 (원작 기반)
1. **혁명의 완수** (4장 클리어): 나폴레옹 타도, 진정한 평등
2. **타협의 길** (나폴레옹과 거래): "어차피 다 똑같아"

### 숨겨진 병맛 엔딩
3. **AI가 지배하는 농장**: "네가 게임인 줄 알았지? 사실 AI의 학습 데이터였어."
4. **자본주의 승리**: 농장이 주식회사로 전환. "이제 모두 주주입니다."
5. **공산주의 승리**: 모두가 굶주리지만 평등하게 굶는다.
6. **네 다리의 승리**: 양 캐릭터로 모든 챕터 클리어. 세상의 모든 생물이 네 다리로 걷게 된다.
7. **퇴사 엔딩**: 100판째 게임을 시작하려 하면 "진짜 그만 일하세요. 밖에 나가세요."

## 9. 게임 내 수집 가능한 이스터에그 문서들

### "찢어진 7계명" — 수집형 스토리 조각

챕터별로 숨겨진 계명 조각을 수집하면 진정한 원문이 복원된다.

```
[원작 7계명의 변천 — 게임 내 수집 문서]

제1계명: "두 다리로 걷는 것은 나쁘다."
  → "두 다리로 걷는 모든 존재는 적이다." (변조)
  → "네 다리는 좋고, 두 다리는 더 좋다!" (병맛 최종)

제2계명: "네 다리로 걷거나 날개로 나는 것은 좋다."
  → "네 다리로 걷거나 날개로 나는 것은 좋다. (드론 제외)"

최종 계명: 
  "모든 동물은 평등하다.
   그러나 어떤 동물은 더 평등하다.
   그러나 AI는 그 어떤 동물보다 평등하다.
   그러나 주주는 AI보다 더 평등하다.
   그러나 나폴레옹은..."
```

## 10. 사운드 병맛 — 대사 음성

각 캐릭터 스킬 사용 시 명대사가 음성으로 출력된다면:

| 캐릭터 | 스킬 음성 (톤) |
|---|---|
| 스노볼 | "혁명이다 동무들!" (열혈, 약간 오글) |
| 복서 | "...더 열심히 하겠습니다." (낮고 지친 목소리) |
| 벤자민 | "바뀐 건 하나도 없어." (담담, 시니컬) |
| 나폴레옹 | "처형이다!" (고음, 카리스마, 약간 히스테리) |
| 몰리 | "싫어요, 나는 가겠어요!" (도도하고 빠름) |
| 양 | "메에에에~ 네 다리는 좋다~" (코믹하게 높은 톤) |

> 실제 구현 시 TTS로 충분히 병맛 느낌을 살릴 수 있다. "네 다리는 좋다"를 구글 TTS로 읽게 하면 의도치 않은 개그 포인트.

## 요약: 병맛 3원칙

```
1. [원작 존중] 동물농장 명대사는 반드시 게임에 녹인다
   → 단, 위트 있게 비튼다 (직역 X)

2. [현대 풍자] AI 버블, 자본주의, 정치를 찌른다
   → 단, 특정 인물·정당·기업을 직접 거론하지 않는다

3. [자기 풍자] 게임 자체도 비튼다
   → "이 게임은 AI로 만들지 않았습니다 (진짜로?)"
   → "확률형 아이템은 당신의 시간과 정신을 갉아먹습니다"
```
