# CHARACTERS.md — 동물농장 캐릭터 & 스프라이트 설계

## 디자인 철학

> **스타듀밸리 × 동물농장 = 귀여운 픽셀로 전하는 잔혹 동화**

- 모든 캐릭터는 **24×24 ~ 32×32 픽셀** 베이스의 스타듀밸리풍 귀여운 동물
- 외형은 아기자기하지만, 스킬명·대사에 원작의 풍자와 어둠을 담는다
- **귀여움과 잔혹함의 괴리**가 게임의 시그니처 무드

## 플레이어블 캐릭터 5인

### 1. 스노볼 (Snowball) — 전략가 돼지 🐖

| 항목 | 내용 |
|---|---|
| 해금 | 기본 |
| 종족 | 돼지 (흰색, 중형) |
| 컬러 팔레트 | `#FFE4D0` 피부, `#FFB6C1` 코, `#8B4513` 발굽 |
| 외형 | 동그란 몸통에 작은 귀, 반짝이는 눈. 목에 빨간 밴다나. |
| 표정 | 지적이고 의욕 넘치는 눈빛. 입가엔 항상 미소. |
| 애니메이션 | 걸을 때 꼬리 살랑. 스킬 사용 시 한쪽 발굽을 든다. |

**스프라이트 시트 (32×32, 4프레임)**
```
[정지] 발굽으로 턱을 괴고 생각하는 포즈. 작은 안경 반짝임.
[이동] 통통 튀는 4프레임 좌우 런. 배가 출렁.
[공격] 두 발굽을 앞으로 내밀며 기합. 입에서 작은 별 발사.
[대시] 빙글 회전하며 이동. 잔상에 설계도 이미지.
```

**성격**: 순수한 혁명가. "모두가 평등한 세상을 만들자" → 아이러니하게도 나폴레옹에게 배신당한다.

| 외형 | 32×32도 부족할 만큼 덩치. 근육질 체형. 항상 붕대 감은 발. |
| 표정 | 진지하고 순박한 눈. 웃을 때도 슬퍼 보인다. |
| 특징 | 등에 실려 있는 작은 건초더미. 칼질 자국 난 갈기. |

**스프라이트 시트 (32×32, 4프레임)**
```
[정지] 네 발로 굳건히 선 자세. 숨결 이펙트.
[이동] 느리지만 묵직한 발걸음. 화면이 살짝 흔들림.
[공격] 앞발굽을 들어올리며 지면 강타.
[대시] 몸을 숙이고 전진. 흙먼지 이펙트.
```

**성격**: "더 열심히 일하겠습니다" — 끝까지 나폴레옹을 믿지만, 그 믿음이 비극이 된다.

## 주요 NPC / 보스

### 농부 존스 — 1장 보스
- **외형**: 48×48 픽셀, 덩치 큰 인간. 찢어진 작업복 + 밀짚모자.
- **팔레트**: 칙칙한 갈색/베이지 계열
- **애니메이션**: 산탄총 들고 느릿느릿. 때리면 욕설 말풍선.
- **병맛**: 쓰러뜨리면 "내 농장 돌려줘!" → 사라지면서 가발이 벗겨진다.

### 프레드릭 — 2장 보스
- **외형**: 48×48 픽셀, 뚱뚱한 인간. 군복 + 위스키 병.
- **애니메이션**: 폭탄 던지기. 폭발에 자기 발이 걸려 넘어지기도.
- **병맛**: 전투 중 "난 자본주의자야!" "아니, 공산주의자!" 왔다갔다.

### 나폴레옹 (보스 버전) — 3장 보스
- **외형**: 플레이어블보다 2배 크기. 황금 왕관 + 빨간 망토.
- **대사**: "모든 동물은 평등하다..." (HP 50% 이하) "그러나 나는 더 평등하다!"
- **패턴**: 플레이어 스킬을 그대로 카피해서 사용.

## 스프라이트 제작 명세

### 기술 스펙
```
포맷: PNG (투명 배경)
해상도: 캐릭터별 24~32px, 보스 48~64px
프레임: 캐릭터당 최소 4프레임 (정지/이동/공격/대시)
스타일: 스타듀밸리 픽셀아트 (윤곽선 + 소프트 셰이딩)
컬러: 각 캐릭터팔레트 6~8색으로 제한 (레트로 감성)
```

### 스프라이트 시트 레이아웃 (예: 32×32)
```
┌────────┬────────┬────────┬────────┐
│ 정지   │ 이동1  │ 이동2  │ 이동3  │
│ idle   │ walk1  │ walk2  │ walk3  │
├────────┼────────┼────────┼────────┤
│ 이동4  │ 공격1  │ 공격2  │ 대시1  │
│ walk4  │ atk1   │ atk2   │ dash1  │
├────────┼────────┼────────┼────────┤
│ 대시2  │ 피격   │ 사망   │ 스킬A  │
│ dash2  │ hurt   │ die    │ skillA │
└────────┴────────┴────────┴────────┘
```

### 현재 PreloadScene.ts → SpriteManager.ts 마이그레이션 계획

현재 `PreloadScene.ts`는 단색 사각형만 생성한다. 이를 다음 단계로 발전:

**Phase A**: Canvas API로 귀여운 픽셀아트 절차적 생성 (외부 에셋 없이)
- 각 캐릭터의 기본 형태를 수학적으로 그리기 (원, 타원, 삼각형 조합)
- 윤곽선 + 그림자 + 하이라이트로 입체감 부여
- 4프레임 애니메이션을 오프셋 변형으로 구현

**Phase B**: 실제 PNG 스프라이트 시트로 교체
- `/public/assets/sprites/characters/snowball.png`
- Aseprite / Piskel / 픽셀 아티스트 협업

**Phase A 구현 예시 (Snowball 돼지)**
```typescript
// SpriteManager.ts - 절차적 픽셀아트 생성기
function generatePigTexture(
  scene: Phaser.Scene,
  key: string,
  palette: { skin: string; nose: string; eye: string },
  size: number = 32
): void {
  const canvas = document.createElement("canvas");
  canvas.width = size * 4; // 4프레임
  canvas.height = size;
  const ctx = canvas.getContext("2d")!;

  for (let frame = 0; frame < 4; frame++) {
    const ox = frame * size;
    // 귀여운 돼지 실루엣
    ctx.fillStyle = palette.skin;
    ctx.beginPath();
    ctx.ellipse(ox + 16, 16, 10, 12, 0, 0, Math.PI * 2);
    ctx.fill();

    // 작은 귀
    ctx.beginPath();
    ctx.ellipse(ox + 8, 6, 4, 5, -0.3, 0, Math.PI * 2);
    ctx.ellipse(ox + 24, 6, 4, 5, 0.3, 0, Math.PI * 2);
    ctx.fill();

    // 코 (동글)
    ctx.fillStyle = palette.nose;
    ctx.beginPath();
    ctx.ellipse(ox + 16, 20, 4, 3, 0, 0, Math.PI * 2);
    ctx.fill();

    // 반짝이는 눈
    ctx.fillStyle = "#FFFFFF";
    ctx.fillRect(ox + 11, 12, 3, 3);
    ctx.fillRect(ox + 21, 12, 3, 3);
    ctx.fillStyle = palette.eye;
    ctx.fillRect(ox + 12, 13, 2, 2);
    ctx.fillRect(ox + 22, 13, 2, 2);

    // 프레임별 변형 (idle: 그대로, walk: 살짝 이동 오프셋 등)
  }

  scene.textures.addSpriteSheet(key, canvas as any, { frameWidth: size, frameHeight: size });
}
```

## 요약: 귀여움과 잔혹함의 공존

```
[외형] 스타듀밸리풍 아기자기 픽셀 동물들
         ↓ 대비
[내용] 동물농장의 배신·폭력·이념 투쟁
         ↓ 양념  
[병맛] 정치 풍자 + 밈 + 현대사회 비틀기
         ↓ 결과
[기억] "귀여운데 묘하게 찝찝한 게임"
```

이 3박자(귀여움 → 잔혹함 → 병맛)가 이 게임의 정체성이다.
